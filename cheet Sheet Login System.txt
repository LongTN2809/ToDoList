1️⃣ Ý tưởng cốt lõi
LOGIN  = tạo session
AUTH   = kiểm tra session

Session dùng để server biết user nào đang đăng nhập.

2️⃣ Flow hoàn chỉnh
User nhập username + password
        ↓
JS gửi fetch POST /login
        ↓
PHP kiểm tra database
        ↓
Nếu đúng → tạo session
$_SESSION['user_id'] = id
        ↓
JS redirect index.html
        ↓
index.html load
        ↓
JS gọi API getUser
        ↓
PHP kiểm tra session
        ↓
Nếu có session → trả user
        ↓
JS hiển thị username
3️⃣ API cần có
🔑 Login
POST /api/user.php?action=login

JS:

fetch("api/user.php?action=login",{
 method:"POST",
 headers:{"Content-Type":"application/json"},
 body: JSON.stringify({username,password})
})

PHP:

if($user && password_verify($password,$user['PASSWORD'])){
    $_SESSION['user_id'] = $user['id'];
}
👤 Get user
GET /api/user.php?action=getUser

PHP:

if(!isset($_SESSION['user_id'])){
    echo json_encode(["success"=>false]);
    exit;
}
